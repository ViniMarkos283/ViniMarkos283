## 🚀 Repositórios em Destaque

### [Projeto Integrador - IntegraKids](https://github.com/Jornada-Da-Inclusao/FrontEnd)
#### Plataforma educacional inclusiva desenvolvida em grupo na Fatec Luigi Papaiz.
Sistema com mini-jogos interativos (números, cores, vogais e memória) voltados para crianças com dificuldades de aprendizado em ambiente escolar. Conta com painel para professores cadastrarem alunos e acompanharem o progresso individual. Projeto hospedado em produção via Firebase Hosting. Fui responsável pelo desenvolvimento front-end em React.js e pela integração com o back-end.

**Tecnologias:** React.js · Java · Spring Boot · Android Studio · MongoDB · Firebase · Render

### [Hackathon - Backend Nazca](https://github.com/ViniMarkos283/Hackathon-backEnd-Nazca)
#### API REST de gestão de conformidade corporativa desenvolvida durante um Hackathon.
Back-end responsável por gerenciar POPs (Procedimentos Operacionais Padrão), treinamentos de colaboradores, emissão automática de alertas e relatórios de conformidade por setor. Inclui jobs agendados para atualização de status, importação de dados via planilha Excel e documentação interativa via Swagger/OpenAPI.

**Tecnologias:** Java · Spring Boot · MySQL · Flyway · Docker · Apache POI · Swagger

### [TCC - FoxWare Web](https://github.com/ViniMarkos283/FoxWare-Web)
#### Site desenvolvido como TCC no curso técnico do SENAI, complementar a um app de realidade aumentada.

Funciona em conjunto com um aplicativo mobile de realidade aumentada desenvolvido em Unity + Vuforia (disponível em repositório separado). O repositório mantém duas branches: a versão original entregue no TCC e uma remake em andamento, incorporando conhecimentos adquiridos na Fatec.

**Tecnologias (Web):** HTML · CSS · JavaScript
